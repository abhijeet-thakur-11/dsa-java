# Data Structures & Algorithms in Java

Welcome to my Java Data Structures & Algorithms (DSA) repository! This repository serves as a compilation of core concepts, algorithmic patterns, and problem-solving techniques implemented in Java.

## 📌 Contents

The repository covers a wide variety of topics, divided into modular Java files:

### 1. ☕ [Basics.java](file:///c:/Users/abhijeet/Desktop/dsa-java-/Basics.java)
Fundamental Java programming concepts and syntax examples:
- **Strings**: Operations (concatenation, `charAt`, substring, comparison) and `StringBuilder` manipulation.
- **Arrays**: 1D and 2D arrays, sorting, and user-input representation.
- **Casting**: Implicit vs. Explicit casting.
- **Math Operators & Input**: Reading inputs via `Scanner` and utilizing mathematical utility functions.
- **Control Flow**: Conditional `if-else` blocks, `switch-case` statements, and loops (`for`, `while`).
- **Exception Handling**: Standard `try-catch` structures.

### 2. 🌀 [recurssion.java](file:///c:/Users/abhijeet/Desktop/dsa-java-/recurssion.java)
A comprehensive collection of recursion problems:
- **Mathematical**: Fibonacci series, calculation of $x^n$ (stack height optimization).
- **Classic Puzzles**: Tower of Hanoi.
- **String Manipulation**: Reversing a string, finding first/last character occurrences, checking if an array is sorted.
- **String Transformations**: Moving all occurrences of a specific character (e.g., 'x') to the end, removing duplicates.
- **Advanced Subsequences**: Finding unique and duplicate subsequences, keypad combinations (permutations).

### 3. 🛡️ [backtacking.java](file:///c:/Users/abhijeet/Desktop/dsa-java-/backtacking.java)
Advanced backtracking solutions for constraint satisfaction problems:
- **Permutations**: Generating all permutations of a string.
- **N-Queens Problem**: Placing $N$ queens on an $N \times N$ chessboard such that no two queens attack each other.
- **Sudoku Solver**: Classic Sudoku backtracking solver using safety checks for rows, columns, and $3 \times 3$ grids.

### 4. 📊 [sort.java](file:///c:/Users/abhijeet/Desktop/dsa-java-/sort.java)
Implementation of foundational sorting algorithms:
- **Bubble Sort**: Simple comparison-based sorting ($O(n^2)$).
- **Selection Sort**: In-place comparison sorting ($O(n^2)$).
- **Merge Sort**: Divide-and-conquer sorting algorithm ($O(n \log n)$).

---

## 🚀 How to Run the Files

Ensure you have the Java Development Kit (JDK) installed. You can compile and run any of the files using your terminal:

```bash
# Compile the Java file
javac Basics.java

# Run the compiled class
java Basics
```
